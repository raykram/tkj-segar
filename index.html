Pekerjaan Tamatan TKJ

Deskripsi.

1

ISP/Network Security

Mengidentifikasi kebutuhan keamanan jaringan, Mendesain sistem keamanan jaringan, Menginstalasi security jaringan, Mengoperasikan security jaringan dan Monitoring keamanan jaringan.

2

Wireless Networking

Merancang, melakukan survey lapangan, Membuat antenna, Menginstalasi jaringan wireless, Mengkonfigurasi peralatan dan Mengoperasikan jaringan wireless.

3

Network Administrator

Mendiagnosis permasalahan perangkat yang tersambung jaringan berbasis luas (Wide Area Network) dan Menganalisa dan memperbaiki kerusakan/kesalahan /tidak bekerjanya koneksi di sistem jaringan.

4

Administrasi Server

Memanage server, Mengatur band width, File sharing, Memonitor server dan Mengatur traffic.

5

Computer Integrator

Merakit komputer, Memperbaiki komputer dan Menginstalasi sistem operasi GUI dan CLI

6

Voip Integrator

Indentifikasi kebutuhan, Merancang jaringan voip dan Instalasi softswitch

7

Linux Administrator

Menginstalasi sistem operasi Linux, Maintenance sistem operasi Linux dan Melakukan virtualisasi

8

Integrator Jaringan

erancang bangun dan menganalisa Wide Area Network, Memasang jaringan lokal, Melakukan instalasi perangkat jaringan berbasis luas (Wide Area Network), Menginstalasi sistem operasi jaringan, Menganalisa memperbaiki kerusakan/kesalahan/tidak bekerjanya koneksi di sistem jaringan dan Merancang web database untuk content server

9

Web Administrator

Konfigurasi web, Perawatan web, Memonitor jaringan, Mengatur traffic dan Melakukan perbaikan atau setting ulang koneksi jaringan WAN.
